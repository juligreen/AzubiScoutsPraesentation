Aufgabe

Erstelle eine Funktion in Python, die die Fakultät einer beliebigen (positiven) Zahl zurückgibt.

Fakultät

Zitat Wikipedia: "Die Fakultät ist in der Mathematik eine Funktion, die einer natürlichen Zahl das Produkt aller natürlichen Zahlen (ohne Null) kleiner und gleich dieser Zahl zuordnet. Sie wird durch ein dem Argument nachgestelltes Ausrufezeichen („!“) abgekürzt."


                1! = 1
                2! = 2 x 1 = 2
                3! = 3 x 2 x 1 = 6
                4! = 4 x 3 x 2 x 1 = 24

Was ist Rekursion?

In der Informatik ist eine Funktion, die sich selbst aufruft rekursiv.

Funktioniert dieser Code?


                def fakultaet(x):
                    return x * fakultaet(x - 1)

Lösung:


                def fakultaet(x):
                    if x < 1:
                        return x * fakultaet(x - 1)
                    else:
                        return 0
